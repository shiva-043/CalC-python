# Basic Calculator - CalC v1.0

## Description
A basic calculator built using python. All basic arithmetic operations are supported. Answer is maintined till exit, so can be used close to a real calculator(useful). At the end all operations made are printed (like a bill).

## Features
1.Menu for selecting operation
2.Answer is carried till exit   (*)
3.Uses files to store history
4.Operations
    1.Addition
    2.Subtraction
    3.Multiplication
    4.Division
    5.Modulus
    6.Nth power
    7.Nth root

## How to run
1.Install Python (Version 3.x)
2.Open calculator (project) folder
3.Run calculator.py
